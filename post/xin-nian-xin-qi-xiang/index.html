新年新气象 | 2022 年终总结

Posted on 2022-12-31，6 min read

工作

加入了一个新家庭，当然先熟悉部门的相关流程规范了，在熟悉了一段时间后，我决定花点时间整理整理当前部门的相关流程规范，毕竟部门的流程规范看更新日期已经是好几年前了，是时候走上时代的潮流了，首先找找找开源主流的编码规范是怎么约定的，对比了几家开源项目的编码约定后，初步决定将规约文档分为如下结构：

  • 编码规约
    • HTML 编码规约
    • CSS 编码规约
    • JavaScript 编码规约
    • TypeScript 编码规约
    • React 编码规约
    • Vue 编码规约
    • Node.js 开发规约
  • 工程规约
    • Git 规约
    • 文档通用规约
    • 前后端协作规约
    • 图片规范
    • 命名规范

并且针对编码规约引入了多个业界流行的 Linter 作为规约文档的配套工具，并根据规约内容定制了 cdu-fabric 规则包，它们包括：

规约 Lint 工具 规则包
《JavaScript 编码规约》
《TypeScript 编码规约》
《React 编码规约》
《Vue 编码规约》
 ESLint eslint-config-cdu
《CSS 编码规约》 stylelint stylelint-config-cdu
《Git 规约》 commitlint commitlint-config-cdu
《文档通用规约》 markdownlint markdownlint-config-cdu

整个编码规约工程大概花了半年时间吧，期间由于工作原因断断续续的进行迭代完善，也认真了去了解了各个lint 工具的官方文档，也从中收获了许多比较推崇最佳编码写法，提高了自己代码质量水平。

下半年由于公司主要参与某省健康码小程序工作，所以利用业余时间和工作时间落地了一个在线远程小程序打包工具，既解决了小程序发版需要本地发版的繁琐，又抹平本地开发环境存在差异的问题，这里说一个小故事，本地开发环境存在差异可能许多人都觉得无关紧要，毕竟现在的打包工具兼容性挺强的，但是我们部门就由于本地打包环境的存在差异的问题，通过本地打包发版小程序，导致小程序上线后出现了重大 bug，并且定位原因还无从点位，重新发版本就好了，这不得不说本地打包存在的危险性了。

生活

工作多年以后，突然发现体重越来越重了，看着身边30+的同事肚子都成了圆滚滚的体型，我也开始害怕了，索性决定开始健身，在花了几个周末好好的去找健身店后，终于找到了一家实惠的健身店，大概900一年，相比之前看的私教健身房动辄几千的年费，现在这家还是挺香，初步打算隔天去健身一次，由于没有报过私教，所以去健身店也只能是做做有氧运动，其它的一些器材还是玩不来，但是我觉得起码运动就行，反正我也不练腹肌，所以就无所谓了，其它的生活变化倒是没什么，毕竟一个人的生活有时候也很枯燥，不是上班就是吃饭。在这期间被同事怂恿报考了软考高级，而且距离考试准备时间也就差不多一个多月，结果也是不言而喻了，我俩都没过，不得不说软考高级还是有点东西的。

驾照

去年一个重要目标就是要把C1驾照拿了，去年过年后便在同事的推荐了报考了最近的一个驾校，然后教练一直说最近科一考试会改革，让我赶紧去考了，不然后面题目变多了就不好考了，于是我便火急火燎的刷了一星期科一，然后便去考试了，好运的是过了，紧接着便是科二了，很多人说科二很容易挂，于是周末没事儿我便去练车，但是周末人也很多，其实练不了几把，练着练着教练又突然跟我说，赶紧去考科二，后面科二要改革，要刷学时，于是我又立马报名了科二考试，虽然我也没练几把，但是就是这么勇，每天晚上下班便去考场练科二，当时有七个人，而且考场上面还得排队，所以每天都是12点才到家，可以说是相当的辛苦，我主要是半坡一直对不准，一直到租车考试也没对准，不过上天眷顾，我在科二考试时一分也没扣，顺利通过，接下来就是愉快的科三了，科三相对比较轻松，毕竟不用对点位了，科三考试第一把挂了，第二把在超车后由于换挡没来及，便紧急刹车，刹车声我都听见了，而且还是下坡路段，当时吓死我了，我以为直接挂了，但是语音没有提示，于是我便二次起步继续考试，在临时停车时由于没按喇叭，被扣了5分，最终也顺利通过了，在12月份中旬的时候，一鼓作气把科四考了，然后顺利拿到小本本了，今年的大目标之一算是完成了。

感情

爱情有时候就是看缘分，在去年四月份愚人节当天，没错，就是愚人节，我向我觉得这辈子就她了的女生表白了，所以说干程序员行业有时候就是很奇葩吧，到现在在一起也半年多了，并且相处也很愉快，虽然中间总是会吵架发脾气什么的，但这不就是小情侣的日常嘛，好了，这里就不分享太多了，避免有撒狗粮嫌疑哈。

展望

每年立flag每年打脸，去年的4个flag有两个被打脸了，今年继续立，不要怕，撸起袖子加油干：

  1. 减肥、减肥、减肥，今年还是要减肥
  2. 上岸国企，宇宙的尽头还是得体制内
  3. 软考中级考过，高级有点难，就不要立了
